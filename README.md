# Ativiza Web Admin

Admin dashboard for Ativiza platform.

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Create `.env` file:
   ```bash
   VITE_API_BASE_URL=http://localhost:3000
   ```

3. Start development server:
   ```bash
   npm run dev
   ```
